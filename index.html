Course Overview

What is Frontend Development?

Frontend development is the process of building the visible parts of a website or web application that users interact with directly. Frontend vs Backend: If a website were a restaurant, the Frontend is the dining area, the menus, and the waiters (what you see and interact with). The Backend is the kitchen, the pantry, and the chefs (behind-the-scenes data processing and storage).

60-Hour Curriculum

Module 1: Introduction to Web Development

30 minutes

We begin by understanding the fundamentals of how the internet operates.

  • Website vs Web App: A website is informative (like a blog), a web app is interactive (like Gmail).
  • Client vs Server: Your browser (Client) requests data; the remote computer (Server) sends it.
  • HTTP vs HTTPS: The protocol for sending data. HTTPS is secure (encrypted).
  • Web Browsers: How Chrome, Firefox, and Safari render code into visuals.
  • How the Internet Works: DNS, IP Addresses, and Packets.

Module 2: HTML Fundamentals

1 hour
  • HTML Structure & Boilerplate
  • Tags and Elements
  • Headings (h1-h6) & Paragraphs
  • Links & Images (Attributes)
  • Ordered & Unordered Lists
  • Tables (tr, th, td)
  • Forms (Inputs, Labels, Buttons)
  • Semantic HTML (header, main, footer)

Example Code:

<!DOCTYPE html>
<html>
<head>
    <title>My First Webpage</title>
</head>
<body>
    <h1>Welcome to Frontend Development</h1>
    <p>This is my first webpage.</p>
    <a href="#">Click Here</a>
</body>
</html>
Explanation: <!DOCTYPE html> declares it's an HTML5 document. The <head> contains metadata like the title. The <body> contains the visible content: a main heading (h1), a paragraph (p), and a clickable link (a).

Module 3: CSS Styling

1 hour

What is CSS? Cascading Style Sheets dictate the look and feel of the HTML elements. We cover Inline, Internal, and External CSS.

  • Colors & Fonts
  • The Box Model (Margin, Border, Padding, Content)
  • Flexbox (1D Layouts)
  • CSS Grid (2D Layouts)
  • Responsive Design & Media Queries
  • CSS Animations & Transitions
  • Hover Effects

Example Code:

body {
    background: #0f172a;
    color: white;
    font-family: Arial;
}

button {
    padding: 10px 20px;
    background: blue;
    color: white;
    border: none;
    border-radius: 8px;
    transition: 0.3s;
}

button:hover {
    background: red;
    transform: scale(1.1);
}
Explanation: This styles the entire body dark blue with white text. It creates a blue button with rounded corners (border-radius). The transition property allows the button to smoothly turn red and grow larger (transform: scale) when the mouse hovers over it.

Module 4: JavaScript Fundamentals

1 hour

What is JavaScript? JS is the programming language of the web that makes pages interactive. It runs directly inside your browser.

  • Variables (let, const) & Data Types
  • Operators & Conditions (if/else)
  • Loops (for, while)
  • Functions
  • Arrays & Objects
  • DOM Manipulation
  • Events & Form Validation
  • Fetch API (Getting data from servers)

Example Code:

function showMessage() {
    alert("Welcome to Frontend Training");
}
<button onclick="showMessage()">Click Me</button>
Explanation: The JS defines a reusable function called showMessage that triggers a browser popup alert. The HTML button uses the onclick event attribute. When clicked, it tells the browser to execute that JavaScript function, showing interaction between HTML and JS.
